# 案件管理システム

Next.js + Supabase + TypeScript で構築された案件管理システムです。

## 機能

- ユーザー認証（Supabase Auth）
- 案件管理（カンバン形式）
- 案件ごとのタスク管理
- 制作状況管理
- ダッシュボード（統計・分析）

## 技術スタック

- **フロントエンド**: Next.js 15, React, TypeScript
- **UI**: Tailwind CSS
- **バックエンド**: Supabase
- **データベース**: PostgreSQL
- **認証**: Supabase Auth
- **デプロイ**: Vercel

## 開発環境

- **パッケージ管理**: npm
- **ビルドツール**: Turbopack
- **コード品質**: ESLint, TypeScript
- **スタイル**: Prettier
